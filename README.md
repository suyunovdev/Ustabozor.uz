# 🛠️ IshTop Platformasi

**IshTop** — ishchilar (ustalar) va buyurtmachilar (offerchilar) o'rtasidagi aloqani soddalashtiruvchi zamonaviy platforma. Bu yerda ustalar o'z xizmatlarini taklif qiladi, buyurtmachilar esa ehtiyojlariga mos mutaxassislarni topadi.

---

## 🌟 Asosiy Imkoniyatlar

### 👨‍🔧 Ustalar (Ishchilar)

- **Ish qidirish:** Mutaxassislik bo'yicha e'lonlarni ko'rish.
- **Buyurtma qabul qilish:** O'ziga ma'qul kelgan ishlarni tanlash va bajarish.
- **Profil:** Bajargan ishlari va reytingini yuritish.

### 💼 Buyurtmachilar (Offerchilar)

- **E'lon berish:** Kerakli ish turi va narxni ko'rsatgan holda buyurtma yaratish.
- **Usta tanlash:** Taklif yuborgan ustalar orasidan eng yaxshisini saralash.
- **Baholash:** Ish yakunida ustaga fikr bildirish va ball berish.

### 🖥️ Admin Panel

- **Analitika:** Platformadagi barcha faolliklarni real vaqt rejimida kuzatish.
- **Statistika:** Kunlik buyurtmalar soni, yangi foydalanuvchilar va daromadlar grafigi.
- **Moderatsiya:** Foydalanuvchilar va e'lonlarni nazorat qilish.

---

## 🚀 Texnologiyalar

Loyiha quyidagi stek yordamida ishlab chiqilgan:

- **Frontend:** React.js / Next.js
- **Backend:** Node.js / Express.js
- **Ma'lumotlar bazasi:** MongoDB (yoki PostgreSQL)
- **Stillash:** Tailwind CSS / Shadcn/UI
- **Analitika:** Chart.js / Recharts (Admin panel uchun)
